# Roadmap and Feature Details Reference

## Primary Documentation Source

**URL:** http://159.195.65.250:3000/docs/intro

This URL serves as the official roadmap and feature details resource for the Spells 'n Squares mod project.

## Local Documentation

The complete documentation from the Docusaurus site has been added to the `docs/` folder, organized into volumes:

- **Volume I: Wizard** - Character origin, house system, stats, skills
- **Volume II: Grimoire** - Spells (charms, transfiguration, curses, dark arts, ancient magic)
- **Volume III: Apothecary** - Brewing mechanics, potions, herbology
- **Volume IV: Menagerie** - Creatures (harmless, competent, dangerous, wizard-killer, spirits/humanoids)
- **Volume V: Armory** - Wandlore, brooms, artifacts, consumables
- **Volume VI: Atlas** - Locations (Hogwarts, London, Ministry, Hogsmeade, global)
- **Volume VII: Chronicle** - Story years 1-7, Fantastic Beasts
- **Volume VIII: SMP Systems** - Economy, law & order, factions, quidditch, minigames
- **Volume IX: Code** - UI/UX, visuals, audio, config
- **Volume X: Expansions** - Uagadou, Mahoutokoro, Cursed Child

## Usage

When working on features or planning development:
1. Refer to the local documentation in the `docs/` folder for detailed specifications
2. Each creature, spell, item, and system has comprehensive implementation details
3. Documentation includes AI behaviors, spawning mechanics, combat systems, taming requirements, and code structure examples
4. Use the documentation as the authoritative source for feature requirements

## Documentation Structure

The documentation follows a detailed chapter-based structure with:
- Overview and classification
- Implementation details (AI, spawning, combat, taming)
- Technical specifications (stats, properties)
- Code structure examples
- Integration points
- Development notes and testing checklists
